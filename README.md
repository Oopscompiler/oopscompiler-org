# OopsCompiler 🚀🧠

OopsCompiler is an interactive coding and learning platform designed to challenge users through timed problem-solving, performance-based evaluation, and progress-driven learning. It blends practice, assessment, and analytics to help users write better and more efficient code.

> 🚧 **Project Status:** Actively Developing (Core Frontend Completed, Features Rolling Out)

---

## 🌟 Key Features

### 🧩 Learning & Practice
- Course-wise structured programming content  
- Practice questions with real-time code execution  
- Multiple attempts allowed to encourage improvement  

### ⏱️ Timed Coding Environment
- **User-defined timers** for practice sessions  
- **Fixed timers** for tests and assessments  
- Optional solution reveal **only after timer completion** - Stopwatch-style tracking for total time spent per question  

### 🧪 Assessments & Certification
- Mandatory **test at the end of each course**
- Scores calculated using:
  - Execution time
  - Code complexity
  - Submission speed
- Earn **completion certificates or badges** after passing  
- Retake tests to improve scores and badges  

### 📊 Code Evaluation & Reports
- Automatic **code complexity analysis**
  - Time Complexity
  - Space Complexity
- Performance comparison with:
  - Optimal solution
  - Previous attempts
- Downloadable **performance reports** after each question  

### 📈 Progress & Competition
- Live user count to show active participants  
- Question-wise performance history  
- Score-based rankings (planned)  

### 🌗 UI & UX
- Clean, distraction-free coding interface  
- Light/Dark mode support  
- Responsive design for all screen sizes
